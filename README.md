# Prerequisites
To get started you need to have the [Git](https://git-scm.com/), NodeJS and MongoDB tools installed on your machine. You also need an editor to work with code like [VSCode](https://code.visualstudio.com/).

### Technologies
- HTML
- CSS
- JavaScript
- NodeJS
    - Express
        - express-handlebars
        - express-session
    - Passport
    - Body-parser
    - Mongoose
    - Bcryptjs
- Bootstrap
- MongoDB
